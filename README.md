# MvcMovie
Introduction to ASP.NET Core MVC on macOS, Linux, or Windows

Prerequisites
Install the following:

- .NET Core 2.1 SDK or later
- Visual Studio Code
- C# for Visual Studio Code

Create a web app with dotnet
From a terminal, run the following commands:

```
mkdir MvcMovie
cd MvcMovie
dotnet new mvc
```
